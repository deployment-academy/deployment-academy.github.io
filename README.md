## Local Development

1. [Install Hugo](https://gohugo.io/getting-started/installing/)
1. Add a new post

    ```shell
    hugo new posts/name-your-post.md
    ```

1. Edit your post using your favorite editor
1. Run it to check your changes locally

    ```shell
    hugo server
    ```
